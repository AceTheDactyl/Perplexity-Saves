# Configuration Directory

Repository configuration and metadata files.

## Files

### Version Control
- **VERSION** - Current version number (semantic versioning)
- **CHANGELOG.md** - Version history and changes
- **FIXES_APPLIED.md** - Bug fixes and corrections log

### Framework Configuration
- **SKILL.md** - UCF skill definition and activation protocol
- **pyproject.toml** - Python project configuration
- **manifest_rrrr.json** - RRRR manifest configuration

### Scripts
- **run.sh** - Execution script for UCF workflows

## Usage

### Check Version
```bash
cat config/VERSION
```

### Review Changelog
```bash
cat config/CHANGELOG.md
```

### Run UCF Workflow
```bash
bash config/run.sh
```
