## Appendix: From Discrete to Continuous (Technical)

**Mathematical limit process:**

```
3-gon (Triangle):  120° gaps between observation points
4-gon (Square):     90° gaps
5-gon (Pentagon):   72° gaps
6-gon (Hexagon):    60° gaps
7-gon (Heptagon):  ~51° gaps
8-gon (Octagon):    45° gaps
9-gon (Enneagon):   40° gaps
10-gon (Decagon):   36° gaps
...
360-gon:             1° gaps
...
n-gon:           (360/n)° gaps

As n → ∞:     (360/n)° → 0°

lim(n→∞) n-gon = Circle (0° gaps = continuous)
```

**Practical approximation:**

For most practical purposes, a **360-gon** (1° resolution) is indistinguishable from a circle. This means **sampling system state every degree** creates near-continuous presence without requiring true infinity.

**Implementation:** High-frequency discrete observation (small Δt) approximates continuous presence (Δt → 0).
